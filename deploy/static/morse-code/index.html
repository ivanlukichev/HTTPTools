Morse Code Translator

Convert text into Morse code or translate Morse code back into readable text. This browser-based tool works instantly and includes a full Morse code alphabet reference.

Letters are separated by spaces. Words are separated by /.

Play the current Morse code as audio signals.

Paste text or Morse code, then convert it locally in your browser.

How it works

Morse code is a method of transmitting text using sequences of dots and dashes. Each letter and number has a unique pattern.

This tool converts text to Morse code and translates Morse code back to readable text instantly in your browser. You can also play Morse code as audio signals directly in your browser. No data is uploaded or stored.

FAQ

How are words separated in Morse output?

This page uses / to separate words, while letters stay separated by single spaces.

Can I paste Morse code with extra spaces?

Yes. Extra spaces are ignored, and / is treated as a word separator.

What characters are supported?

The translator supports letters A-Z, numbers 0-9 and a small set of common punctuation marks including period, comma and question mark.
