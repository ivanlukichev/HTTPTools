How it works

This tool compares two line-based lists and finds shared or unique items. It can also trim spaces, ignore empty lines and deduplicate entries before comparing.

All processing happens locally in your browser. No list data is uploaded or stored.

List A:

banana

List B:

grape

FAQ

Can I compare keyword lists or exports?

Yes. The tool is useful for tags, keyword lists, records and any line-based text export.

Can I remove duplicates before comparing?

Yes. Enable Remove duplicates to keep only the first occurrence of each normalized item in each list.

Does this tool upload my list data?

No. Everything stays in your browser.
